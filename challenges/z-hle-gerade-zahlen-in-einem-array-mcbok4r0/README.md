# Zähle gerade Zahlen in einem Array

    ## 📝 Beschreibung
    Schreibe eine C#-Funktion, die ein Array von Ganzzahlen entgegennimmt und die Anzahl der geraden Zahlen darin zurückgibt.

**Beispiele:**

Eingabe: [1, 2, 3, 4, 5, 6]  
Ausgabe: 3

Eingabe: [7, 9, 13]  
Ausgabe: 0

Eingabe: [2, 4, 6, 8]  
Ausgabe: 4

    ## 🎯 Schwierigkeit
